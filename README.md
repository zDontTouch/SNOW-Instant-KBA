# SNOW-Express-KBA
The Express KBA is a script tool for the SAP Integrated Support Environment which adds a little draggable box where users can directly add KBAs to open cases without the regular steps.
<p align="center">
  <img src="https://github.com/zDontTouch/SNOW-Instant-KBA/blob/bc0252111cdb35789febaf56141191d1adfc8614/screenshots/Screenshot%202026-05-13%20115545.png" />
</p>
Normally, users must click on the "Solution" menu, then click on the "Attach KBA" option from the dropdown to open the KBA list popup page. Then users must paste the KBA ID and click on Attack before being able to close the popup.
With this extension, users can simply type the ID of the KBA in the textbox and hit enter to add the KBA through a script in a background tab. Users can even paste the whole ID+Title of the KBA, as the tool will automaticaly trim the ID.

The extension also has a built-in bookmark list, which allows users to store their most used KBAs in a list, where the KBAs can be attached with a single click:
<p align="center">
  <img src="https://github.com/zDontTouch/SNOW-Instant-KBA/blob/bc0252111cdb35789febaf56141191d1adfc8614/screenshots/Express%20KBA%202.0%20Bookmarks.png" />
</p>

In addition to the bookmarks, the script also includes a suggestion (💡) button, which lists KBAs suggested for the case which is currently open. Aside from being able to attach the KBA by clicking on it, the list also allows the users to open the KBA in a new browser tab, and shows the score for the accuracy of the KBA.
